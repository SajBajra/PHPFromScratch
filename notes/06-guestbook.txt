As a small project, I turned the file reading and writing examples into a simple "guestbook" page. The guestbook uses a single PHP file that both shows a form and lists previous entries. When I submit the form, the script validates that the message is not empty, adds a timestamp, and appends the line to a `guestbook.txt` file using `file_put_contents` with append and locking flags. On every request it also reads the same file with `file()` and displays each line (newest first) as a list item in HTML. This helped me see how forms, validation, and basic file I/O can work together to make a tiny but complete feature without a database.
