Today I started learning how to use cookies in PHP. I built a very small page that lets me enter my name and save it into a cookie using the `setcookie` function. On each request, I read the value back from the `$_COOKIE` superglobal and show a "Welcome back" message if it exists. I also created a separate page that clears the cookie by calling `setcookie` with an expiration time in the past. Finally, I built a "theme preference" example where I store either "light" or "dark" in a `theme` cookie and then change the page styles based on that value. This showed me how cookies can remember user preferences across multiple visits, and how they differ from sessions (cookies live in the browser, while sessions are stored on the server and referenced by a session cookie).
