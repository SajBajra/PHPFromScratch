Today I started learning how to work with files in PHP. I created a simple page that lets me type a message into a form and then append it to a text file using `file_put_contents` with the `FILE_APPEND` and `LOCK_EX` flags. I store the file under a `data` directory in my project and make sure that the directory exists before writing. I also built a second page that reads the same file using `file()` and displays each line as a list item in HTML. These small examples helped me see how to write to and read from plain text files, and how this can be used as a very simple way to store data before moving on to a real database.
