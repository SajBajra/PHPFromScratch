Testing with PHPUnit

I added a small helpers file (code/helpers.php) with a validate_email() function that returns true for valid emails and false otherwise. Then I added PHPUnit as a dev dependency in composer.json and created tests/HelpersTest.php that extends PHPUnit's TestCase and tests validate_email with a few valid and invalid examples. After running "composer install" (or "composer update") from the project root, you can run the tests with "vendor/bin/phpunit tests" or "php vendor/bin/phpunit tests". This introduces unit testing so we can check that helper logic works without hitting the browser or database.
