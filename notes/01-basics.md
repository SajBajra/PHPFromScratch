## PHP Basics (syntax, variables, types)

- PHP files typically start with `<?php`.
- Statements end with a semicolon `;`.
- Variables are prefixed with `$` (for example, `$name`, `$age`).
- Common scalar types: integers, floats, strings, booleans.
- Use `echo` to output text, and `PHP_EOL` for a newline in CLI scripts.

### PHP vs Python vs Java (quick comparison)

- **Variables**
  - PHP: `$age = 20;`
  - Python: `age = 20`
  - Java: `int age = 20;`

- **Printing**
  - PHP: `echo "Hello";`
  - Python: `print("Hello")`
  - Java: `System.out.println("Hello");`

- **String interpolation**
  - PHP: `$name = "Saj"; echo "Hello, $name";`
  - Python: `name = "Saj"; print(f"Hello, {name}")`
  - Java: `String name = "Saj"; System.out.println("Hello, " + name);`
