Simple blog

I built a minimal blog using the same php_learn database. The posts table has id, title, body (TEXT), and created_at. 27-blog-list.php creates the table if needed, lists all posts (newest first) with links to view each one, and uses the shared header/footer layout. 27-blog-view.php loads a single post by id and displays title, date, and body (with nl2br for line breaks). 27-blog-add.php shows a form for title and body; on POST it validates (title and body required, title max 255 chars), inserts the row, sets a flash message, and redirects to the list. This reused the same patterns: PDO, prepared statements, session flash, validation, and the layout includes.
