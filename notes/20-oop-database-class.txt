Basic OOP: Database class

I added a very small Database class (code/Database.php) as an introduction to OOP. The class takes the config array from db-config.php in its constructor and creates a PDO instance with error mode set to exceptions and default fetch mode to associative arrays. It stores the PDO object in a private property and exposes a getPdo() method. Then I updated 17-pdo-basics.php to require Database.php, create a new Database with the config, and call getPdo() to get the PDO object. This keeps the connection setup logic in one place and shows how a class can wrap related behaviour.
